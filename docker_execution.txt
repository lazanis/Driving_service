BEFORE:
remove pkg_resources==0.0.0 from requirements.txt
sudo chmod -R 755 /var/lib/mysql/
***change uvicorn app host and dbworker address from 'localhost' to '127.0.0.1'

DOCKER:
1. maker Dockerfile
2. (sudo) docker image build -t NAME(driving-service-image) .
3. (sudo) docker image ls -> list all images
4. (sudo) docker container ls -> list all containers
# 5. (sudo) docker run --network="host" -p 7777:7777 NAME(driving-service-image) -> first port is host, second port is inside container (actual app port)
5. (sudo) docker run --network="host" NAME(driving-service-image)
6. (sudo) docker stop CONTAINER_ID
7. (sudo) docker system prune

DOCKER-COMPOSE:
1. move docker-compose.yml file to a folder above Driving_service and Driving_service_interface
2. (sudo) docker-compose run driving-service-interface-image -> for enabling interactive mode for driving-service-interface
